*Smart Attendance Management System(SAM app)*
▶️ an app that Implements QR technology and has created a mobile-based system that ensures secure, real-time, and verifiable attendance tracking through QR code scanning and digital record storage. 
This is to replace the old system used to record attendance using signing sheets .With this new system
Lecturers and students are able to perform this task with ease using  their mobile phones and also reduces risks of losing track of attendance and also storage of records.
